We welcome contributions! Whether you found a typo in a diagram or optimized a driver info 20% faster code.

## How to Contribute
1.  **Fork** the repository.
2.  **Create a Branch**: `git checkout -b feature/better-laser-driver`.
3.  **Commit**: Stick to the "Professional Code Style" (Use types, docstrings, and comments).
4.  **Push & PR**: Send it in!

## Style Guide
- **Code**: Python files must have a header explaining the goal.
- **Hardware**: Always list the wiring at the top of the file.
- **Safety**: If a sensor is dangerous (Laser, Relay, Flame), add a warning at the top.
